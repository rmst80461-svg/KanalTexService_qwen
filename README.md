# Система управления услугами по откачке септиков в Ярцево

## Описание проекта
Веб-приложение для управления услугами по откачке септиков и выгребных ям в Ярцево, включающее:
- Telegram-бота для приема заказов
- Админ-панель для управления заказами и исполнителями
- Базу данных клиентов с возможностью идентификации новых и постоянных клиентов
- Мобильный интерфейс для администратора

## Архитектура
- Frontend: HTML/CSS/JS + Bootstrap
- Backend: Python Flask
- Database: SQLite
- Telegram Bot: python-telegram-bot
- Deployment: Docker (опционально)

## Структура проекта
```
project/
├── app.py                 # Основной бэкенд
├── bot.py                 # Telegram-бот
├── database.py            # Управление базой данных
├── templates/             # HTML шаблоны
├── static/                # CSS, JS, изображения
├── requirements.txt       # Зависимости
└── README.md              # Документация
```

## Функциональность
### Для клиентов:
- Выбор услуги через Telegram-бота
- Просмотр списка доступных услуг
- Оформление заявки на вызов спецтехники
- Получение уведомлений о статусе заявки

### Для администратора:
- Мобильная админ-панель
- Распределение заказов между исполнителями
- Просмотр истории заказов
- Идентификация новых/постоянных клиентов
- Управление статусами заявок

### Для исполнителей:
- Получение заданий от администратора
- Обновление статуса выполнения
- Отчетность о проделанной работе
