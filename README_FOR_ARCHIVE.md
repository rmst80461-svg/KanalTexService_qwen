# Проект "КаналТехСервис" - Система управления заявками

Добро пожаловать в архив проекта "КаналТехСервис" - полнофункционального решения для управления заявками на ассенизаторские услуги в г. Ярцево.

## Описание проекта

Это веб-приложение сочетает в себе Telegram-бота и веб-админку для эффективного управления заявками клиентов. Клиенты могут подавать заявки через Telegram-бота, а администраторы - управлять этими заявками через удобный веб-интерфейс.

## Содержимое архива

- **main.py** - основной файл приложения (Telegram-бот + Flask-админка)
- **requirements.txt** - зависимости Python
- **create_admin_password.py** - утилита для генерации хеша пароля администратора
- **templates/** - папка с HTML-шаблонами (login.html, dashboard.html)
- **Документация** - файлы README.md, DEPLOYMENT_GUIDE.md, INSTALLATION_GUIDE.md, PROJECT_SUMMARY.md, FILES_DESCRIPTION.md
- **Procfile** - файл конфигурации для облачных платформ
- **.gitignore** - правила игнорирования файлов для Git

## Функциональность

### Telegram-бот
- Подача заявок на ассенизаторские услуги
- FSM-машина для сбора данных (ФИО, адрес, тип услуги, телефон)
- Защита от спама (не более 1 заявки в 24 часа)
- Команда /status для проверки статуса заявки

### Веб-админка
- Авторизация с использованием логина/пароля
- Просмотр всех заявок в табличном виде
- Изменение статуса заявок
- Добавление комментариев
- Экспорт в CSV

## Установка и запуск

1. Установите зависимости: `pip install -r requirements.txt`
2. Создайте файл .env с необходимыми переменными окружения
3. Сгенерируйте хеш пароля: `python create_admin_password.py your_password`
4. Запустите приложение: `python main.py`

Для подробной информации смотрите файл INSTALLATION_GUIDE.md
