# Описание файлов проекта "КаналТехСервис"

## Структура проекта
```
├── main.py                           # Основной файл приложения (Telegram-бот + Flask-админка)
├── requirements.txt                  # Зависимости Python
├── create_admin_password.py         # Утилита для генерации хеша пароля администратора
├── .gitignore                       # Правила игнорирования файлов для Git
├── Procfile                         # Файл конфигурации для Heroku/BotHost
├── templates/                       # Папка с HTML-шаблонами
│   ├── login.html                   # Страница входа в админку
│   └── dashboard.html               # Главная страница админки
├── database_schema.md               # Документация по структуре БД
├── README.md                        # Основная документация
├── DEPLOYMENT_GUIDE.md             # Инструкция по развертыванию
├── INSTALLATION_GUIDE.md           # Инструкция по установке проекта
├── PROJECT_SUMMARY.md               # Полный обзор проекта
└── services.html                    # Страница с описанием услуг
```

## Подробное описание файлов

### main.py
Основной файл приложения, который объединяет в себе Telegram-бота и веб-админку. Содержит:
- FSM-машину состояний для обработки заявок через Telegram-бота
- Маршруты Flask для веб-админки
- Работу с базой данных SQLite
- Отправку уведомлений пользователям
- Обработку статусов заявок

### requirements.txt
Файл зависимостей Python, содержащий список всех необходимых пакетов:
- aiogram для работы с Telegram API
- Flask для веб-части
- Flask-Login для аутентификации
- pandas для экспорта данных
- python-dotenv для работы с переменными окружения
- и другие зависимости

### create_admin_password.py
Скрипт для генерации хеша пароля администратора. Используется для безопасного хранения паролей в переменных окружения.

### .gitignore
Файл с правилами игнорирования для Git, исключающий конфиденциальные и временные файлы:
- Файлы переменных окружения (.env)
- Виртуальные окружения Python
- Логи и временные файлы
- Файлы IDE

### Procfile
Файл конфигурации для облачных платформ (Heroku, BotHost), определяющий команду запуска приложения.

### templates/login.html
HTML-шаблон страницы входа в административную панель с формой аутентификации и Bootstrap-стилями.

### templates/dashboard.html
HTML-шаблон главной страницы админки с таблицей заявок, возможностью изменения статусов и фильтрации, реализованный с использованием Bootstrap.

### database_schema.md
Документация по структуре базы данных, описывающая все таблицы и их поля для будущего масштабирования.

### README.md
Основная документация проекта с описанием функциональности, архитектуры и технологий.

### DEPLOYMENT_GUIDE.md
Подробная инструкция по развертыванию приложения на BotHost.ru.

### INSTALLATION_GUIDE.md
Пошаговая инструкция по установке и запуску проекта.

### PROJECT_SUMMARY.md
Полный обзор проекта с техническими деталями.

### services.html
Страница с подробным описанием предоставляемых услуг для клиентов.
