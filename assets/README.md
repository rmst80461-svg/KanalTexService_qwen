# Assets Directory

## Логотип КаналТехСервис

Поместите сюда логотип вашей компании:

- **Имя файла:** `logo.jpg` (или `logo.png`)
- **Рекомендуемый размер:** 800x600 пикселей (или пропорционально)
- **Формат:** JPG, PNG

Логотип будет отображаться при запуске бота командой `/start`.

### Как добавить логотип:

1. Подготовьте изображение логотипа
2. Сохраните его как `logo.jpg`
3. Загрузите в эту директорию (`assets/`)
4. Перезапустите бота

Если файл `logo.jpg` не найден, бот будет работать без логотипа (только текст).
