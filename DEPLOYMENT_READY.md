
### 2️⃣ ПОЛУЧИТЬ КЛЮЧИ (2 минуты)

```
🔑 BOT_TOKEN     от @BotFather
🔑 ADMIN_ID      от @userinfobot
```

### 3️⃣ НАСТРОИТЬ BOTHOST (5 минут)

```
⚙️ Settings → GitHub → Connect Repository
⚙️ Settings → Environment Variables → Добавить BOT_TOKEN, ADMIN_IDS, PORT
⚙️ Settings → Deployment → Deploy
```

### 4️⃣ ПРОВЕРИТЬ (2 минуты)

```
✅ Telegram → Найти бот → /start
✅ Проверить меню и кнопки
✅ Отправить /admin (если админ)
```
